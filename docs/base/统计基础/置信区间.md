# 置信区间

例子:

歌曲的时长服从正态分布X~N(μ, \\(σ^2\\))，\\(σ^2=1\\);手机里有100 首歌曲，这100 首歌曲的平均时长是4 分钟，求μ的置信水平为 95%的置信区间

样本均值=4

样本容量=100

总体标准差=1

α=0.05

$$
(\frac{}{X}±\frac{σ}{\sqrt{n}}Z_(α/2))=(4±\frac{1}{\sqrt{100}}×1.96)=(3.804, 4.196)
$$

(3.804, 4.196)属于那些包含真值的区间的可信程度为 95%；(3.804, 4.196)包含真值这一陈述的可信度为 95%。
